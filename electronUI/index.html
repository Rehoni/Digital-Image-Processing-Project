基于Python Scikit-image图形库的Electron简单GUI实现

安装和使用

根据此指导安装本应用，Electron快速开始应用是与此应用相同的基本简单框架。 安装时请确保你安装好了Git，Node.js和Python😉 


$ git clone https://github.com/electron/electron-quick-start
$ cd electron-quick-start
$ npm install
$ electron .

关于代码

本应用基于Electron技术实现GUI前端交互设计，基于Python的Scikit-image 图形库实现了后端的功能业务，采用MVVC前后端分离，用不同语言设计。
